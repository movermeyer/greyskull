Greyskull: An NTrack Implementation
===================================



Setup Instructions
------------------

Greyskull is configured using the following environment variables:

| Name                            | Description                                | Default
+
| ``GREYSKULL_EXTERNAL_PORT``       || 80
+
| ``GREYSKULL_MEMCACHED_ENDPOINTS`` || '127.0.0.1:11211'
